Using R to analyse the German Federal Election

June 29, 2018

This blog post is about the last German Federal Election that took place on 24 September of 2017. It will not be an exhaustive analysis about the results. I’m only interested in mapping the share of vote that each party that is represented in Parliament (i.e. Bundestag) had in each one of the 16 German states. The first step is to load the packages that I need to make this mapping possible in R.

library(tidyverse)
library(rgeos)
library(rgdal)
library(sf)
library(broom)
library(readxl)
library(viridis)
library(cartogram)
library(maps)
library(here)

Next, I need to get the file with the German Election results. You can get it from the Federal Returning Officer - https://www.bundeswahlleiter.de/en/bundestagswahlen/2017/ergebnisse.html

After I opened it in Excel, I saved it as .xlsx file in my working directory of R. Then I wrote this code to load the file.

germany_elections_untidy <- read_excel(here("btw17_kerg.xlsx"), skip = 5) # skip the first five rows of the excel file

In the next step, I needed to clean this file. My goal was to have the columns with the German states and the absolute and percentage values of each one of the 6 parties (i.e. CDU/CSU, SPD, FDP, AfD, Die Linke, and Gruene) represented in the Bundestag for each German State.

even_columns<-seq(20,190,2)
odd_columns<-seq(3,87,2)

germany_tidy <- germany_elections_untidy %>%
  select(1:2, even_columns) %>%
  slice(c(14, 22, 30, 62, 66, 78, 89, 103, 169, 187, 211, 221, 238, 286, 326, 332)) %>%
  rename(state = Gebiet, CDU = X__14, SPD = X__17, DieLinke = X__20, 
         Gruenen = X__23, CSU = X__26, FDP = X__29, AfD = X__32, Piraten = X__35, NPD = X__38, FW = X__41, PMUT = X__44, ODP = X__47, PArbeit = X__50, BP = X__53, ADV = X__56, 
         PdV = X__59, MLPD = X__62, BS = X__65, SP = X__68, DR = X__71, ADD = X__74, AMTS = X__77, BergP = X__80, BG = X__83, DBewe = X__86, DKP = X__89, DMitte = X__92, Grauen = X__95, UrbaneHipHop = X__98, MPartei = X__101, MWelt = X__104, PH = X__107, PGesund = X__110, VPartei = X__113, BCD = X__116, Einheit = X__119, Violetten = X__122, FamilienP = X__125, FPDF = X__128, Mieter = X__131, NLiberale = X__134, UBD = X__137, U = X__140) %>% # rename columns' names
  select(-odd_columns) %>% # delete columns
  mutate_at(vars(3:45), as.numeric) %>% # mutate variables from the 3rd to the 45th column to numeric
  mutate_at(vars(1), as.character) %>% # mutate 1st column to character to make
  mutate(CDU_CSU = rowSums(.[c("CDU", "CSU")], na.rm = TRUE), 
         CDU_CSU_per = CDU_CSU / rowSums(.[3:45], na.rm = TRUE) * 100,
         SPD_perc = SPD / rowSums(.[3:45], na.rm = TRUE) * 100,
         DieLinke_perc = DieLinke / rowSums(.[3:45], na.rm = TRUE) * 100,
         Gruenen_perc = Gruenen / rowSums(.[3:45], na.rm = TRUE) * 100,
         FDP_perc = FDP / rowSums(.[3:45], na.rm = TRUE) * 100,
         AfD_perc = AfD / rowSums(.[3:45], na.rm = TRUE) * 100) %>% # create the CDU/CSU variable and all the percentage values of each party for each state 
  select(Nr, state, CDU_CSU_per,SPD_perc,DieLinke_perc, Gruenen_perc, FDP_perc, AfD_perc, CDU_CSU,
         everything(), -10:-45)

However, I still need to recode the values of the Nr variable to become compatible with the shapefile of Germany that I will load afterwards.

Nr_recode <- c("10" = "11", "9" = "1", "1" = "14", "4" = "4", "13" = "7", "8" = "0","5" = "9", "6" = "6", "12" = "3", "7" = "10", "14" = "12", "11" = "2", "15" = "13","2" = "5", "16" = "15", "3" = "8") # "old value" = "new value"
germany_tidy$Nr <- Nr_recode[germany_tidy$Nr] # assign the new values to the Nr variable

However, my goal is to create a map of Germany with the results of the German Federal Elections. For that we need to load a shapefile of Germany. I have downloaded the shapefile and put it in my working directory. After that I use the function readOGR() from the rgdal package to load the file in R.

germany <- readOGR(here(layer = "DEU_adm1.shp"), use_iconv = TRUE, encoding = "UTF-8")

Ok, now I have one shape file with the map of Germany and one data frame with the German Federal Elections results. So, they are separated from each other, but I must join them so that I can map the elections results in the map of Germany. But before joining them, I should prepare this shape file make possible the joining with the German Federal election dataframe. Therefore, I create a new variable called germany_states.

germany_states <- germany@data$NAME_1
germany@data$germany_states <- germany_states # prepare the shape file with the map of germany to make it possible to join the dataframe of german elections

Then, I have to filter the state variable present in the the German Federal election dataframe with the correspondent variable created before in the shape file.

elections_data <- germany_tidy %>% filter(state %in% germany_states) # filter data for germany states

Now, everything is ready to join both datasets, the shape and the German Federal Election dataframe called elections_data

Let’s join both.

germany@data <- germany@data %>% left_join(elections_data, 
                                           by = c("germany_states" 
                                                  = "state")) # join datasets
rownames(germany@data) <- germany@data$germany_states # important step for the indexes to join

Finally, I just have to transform the previous object in a dataframe. I need to make this, because I cannot plot shape files, just dataframe. For this, Ill use the tidy() function from the broom package.

germany_dataframe <- tidy(germany) %>% left_join(., germany@data, by = c("id" = "Nr")) # transform the previous object in a dataframe
# plot
x_final <- x_final <- ggplot(germany_dataframe, aes(x = long, y = lat, group = group)) +
  geom_polygon(aes(fill=CDU_CSU_per), color = "grey50", data =filter(germany_dataframe, !NAME_1 %in% c("Berlin", "Bremen")))+
  geom_polygon(aes(fill=CDU_CSU_per), color = "grey50", data =filter(germany_dataframe, NAME_1 %in%  c("Berlin", "Bremen")))+
  theme_minimal() +
  coord_equal()
x_final
